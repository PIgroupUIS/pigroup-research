Resumen

An automated system for classifying cocoa bean fermentation levels using computer vision and machine learning techniques. This research addresses the critical need for quality control in cocoa production by developing an intelligent system that can accurately assess fermentation levels. Our approach combines advanced image processing algorithms with deep learning models to analyze visual characteristics of cocoa beans. The system provides farmers and producers with an objective, consistent method for quality assessment, potentially improving product quality and market value.
